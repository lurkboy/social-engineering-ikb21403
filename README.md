## Exercise 1: Phishing Attack

### 1. Description
Phishing is a social engineering attack where attackers send fake emails pretending to be trusted organizations to trick victims into revealing sensitive information such as usernames, passwords, or banking details.

### 2. How the Attack Works
1. Attacker creates a fake email (bank, university, delivery service)
2. Email contains a malicious link
3. Victim clicks the link
4. Victim enters personal information on a fake website
5. Attacker steals the information

### 3. Example Scenario
A user receives an email claiming their bank account is locked and is asked to log in immediately using a provided link.

### 4. Impact
- Credential theft
- Financial loss
- Identity theft

### 5. Prevention
- Check sender email address
- Do not click suspicious links
- Enable multi-factor authentication

## Exercise 2: Smishing Attack

### 1. Description
Smishing is a type of phishing attack conducted through SMS or messaging applications where attackers trick victims into clicking malicious links or sharing sensitive information.

### 2. How the Attack Works
1. Attacker sends a fake SMS
2. Message claims urgent action is required
3. Victim clicks the link
4. Victim is redirected to a fake website

### 3. Example Scenario
A victim receives an SMS stating that their parcel delivery failed and must be rescheduled by clicking a link.

### 4. Impact
- Malware infection
- Data theft
- Financial scams

### 5. Prevention
- Do not click unknown links
- Verify messages with official sources
- Use spam filters and security awareness
